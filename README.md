## 학습 자세

1. **AI 도구 적극 활용**: ChatGPT·Claude로 코드·표·시나리오 생성을 자동화. 단, **AI 사용을 숨기지 않는다** (기말 프로젝트는 AI 활용 투명성 보고 의무).
2. **진실성 우선**: AI 환각으로 만들어진 가짜 통계·인물·인용은 보고서에 들어가면 안 된다. 출처 검증 습관이 핵심.
3. **도구 충돌이 의사결정 지점**: 13주차 워크숍의 핵심 통찰 — 5개 도구가 동의하면 평범한 결정. **충돌하는 지점이 진짜 사고를 요구한다**.
