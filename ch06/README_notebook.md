# 6주차: AI 기반 환경 스캐닝

## 수업 목표

이번 주차의 목표는 복잡한 분석기법을 배우는 것이 아니라, AI를 활용해 기획자가 주변 변화를 빠르게 읽는 방법을 익히는 것이다.

학생은 수업 후 다음을 할 수 있어야 한다.

1. 환경 스캐닝이 왜 필요한지 설명할 수 있다.
2. 뉴스 제목과 짧은 문장에서 반복되는 이슈를 찾을 수 있다.
3. 약신호를 기회와 위험으로 나누어 해석할 수 있다.
4. AI에게 분석 코드를 요청하고, 결과를 기획 관점에서 해석할 수 있다.

## 핵심 개념

### 환경 스캐닝

환경 스캐닝은 계획을 세우기 전에 바깥 변화를 살피는 일이다. 정책, 기술, 시장, 사회 분위기, 경쟁자 움직임을 넓게 훑어보고 "앞으로 문제가 될 수 있는 변화"를 찾는다.

### 약신호

약신호는 아직 작지만 앞으로 중요해질 수 있는 변화의 조짐이다. 뉴스가 갑자기 늘거나, 특정 단어가 반복되거나, 사람들의 반응이 빠르게 바뀌면 약신호일 수 있다.

### 기획자의 질문

- 이 변화는 우리에게 기회인가, 위험인가?
- 아직 작지만 커질 가능성이 있는가?
- 지금 관찰만 할 것인가, 바로 대응할 것인가?

## 바이브 코딩 방식

학생은 코드를 처음부터 외워서 작성하지 않는다. 대신 아래 순서로 진행한다.

1. 분석하고 싶은 일을 자연어로 쓴다.
2. AI에게 Python 코드를 요청한다.
3. 코드를 실행한다.
4. 오류가 나면 오류 메시지를 AI에게 다시 보여준다.
5. 결과를 표와 짧은 문장으로 해석한다.

예시 프롬프트:

```text
다음 뉴스 제목 목록에서 자주 등장하는 키워드를 세고,
처음 절반과 나중 절반의 언급량을 비교해서
최근에 증가한 이슈를 찾는 Python 코드를 만들어줘.
결과는 표와 막대그래프로 보여줘.
```
