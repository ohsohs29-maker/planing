# 7주차: 이해관계자 분석과 정책 네트워크

## 수업 목표

이번 주차의 목표는 정책 네트워크 이론을 깊게 다루는 것이 아니라, AI를 활용해 이해관계자를 빠르게 정리하고 기획 관점으로 해석하는 방법을 익히는 것이다.

학생은 수업 후 다음을 할 수 있어야 한다.

1. 이해관계자가 무엇이고 왜 식별해야 하는지 설명할 수 있다.
2. 영향력-관심도 매트릭스로 이해관계자를 4개 분면으로 나눌 수 있다.
3. 누가 누구에게 영향을 주는지 간단한 네트워크로 그릴 수 있다.
4. AI에게 분석 코드를 요청하고, 결과를 기획 관점에서 해석할 수 있다.

## 핵심 개념

### 이해관계자

이해관계자는 정책이나 사업의 **결과에 영향을 받거나, 결과에 영향을 줄 수 있는 사람·집단**이다. 정부 부처, 시민 단체, 기업, 주민, 언론, 전문가 등이 모두 포함될 수 있다.

### 영향력-관심도 매트릭스 (Power-Interest Matrix)

이해관계자를 **영향력(Power)**과 **관심도(Interest)** 두 축으로 나누어 4분면에 배치한다.

```
영향력 높음
   │
   │  관리(Manage)        ★ 핵심 관계자
   │  - 정기 보고            (Key Player)
   │  - 의사결정 참여      - 긴밀히 협력
   │
   ├──────────────┼──────────────
   │
   │  최소 관심          정보 제공
   │  (Monitor)         (Keep Informed)
   │  - 동향만 살핌      - 소식 전달
   │
   └────────────────────── 관심도 높음
```

### 정책 네트워크의 직관

이해관계자는 따로 떨어져 있지 않다. **누가 누구에게 영향을 주는지**를 선으로 이으면 네트워크가 된다.
중심에 가까운 사람은 작은 행동에도 파급력이 크다. 가장자리에 있는 사람은 직접 설득하기 어렵다.

### 기획자의 질문

- 이 정책에 누가 영향을 받는가?
- 누구를 먼저 설득해야 하는가?
- 빠뜨린 이해관계자가 있는가?

## 바이브 코딩 방식

학생은 코드를 처음부터 외워서 작성하지 않는다. 대신 아래 순서로 진행한다.

1. 분석하고 싶은 일을 자연어로 쓴다.
2. AI에게 Python 코드를 요청한다.
3. 코드를 실행한다.
4. 오류가 나면 오류 메시지를 AI에게 다시 보여준다.
5. 결과를 표와 짧은 문장으로 해석한다.

예시 프롬프트:

```text
다음 이해관계자 목록을 영향력-관심도 매트릭스로 표시하는
Python 코드를 만들어줘.
영향력과 관심도는 0~10 점수이고, 점은 흑백으로,
이름을 옆에 표시해줘.
```

```text
이해관계자들이 서로 누구에게 영향을 주는지 표로 받아서
네트워크 그래프를 그리는 Python 코드를 만들어줘.
중심성이 높은 사람을 표로 정리해줘.
```
