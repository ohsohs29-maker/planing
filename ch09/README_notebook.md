# 11주차: 몬테카를로 시뮬레이션

## 수업 목표

이번 주차의 목표는 통계 이론을 배우는 것이 아니라, **불확실한 입력값을 분포로 표현하고 수천 번 굴려서 결과 분포를 얻는** 사고방식을 익히는 것이다.

학생은 수업 후 다음을 할 수 있어야 한다.

1. 점 추정과 확률적 추정의 차이를 한 문장으로 설명할 수 있다.
2. 정규·삼각·균등·로그정규 분포의 쓰임을 구분할 수 있다.
3. 매출·비용·이익을 분포로 표현해 시뮬레이션할 수 있다.
4. 결과 분포를 평균·90% 구간·손실 확률로 해석할 수 있다.
5. 토네이도 차트로 핵심 리스크 변수를 식별할 수 있다.
6. 시나리오 플래닝(10주차)과 시뮬레이션을 통합할 수 있다.

## 핵심 개념

### 점 추정의 한계

"내년 매출은 10억"이라고 답하면 깔끔하지만 **틀릴 가능성**을 숨긴다. 몬테카를로는 입력값을 분포로 두고 1만 번 굴려서 결과 분포를 만든다.

### 분포 4종 매뉴얼

| 분포 | 언제 쓰는가 | 입력 |
|------|-----------|------|
| **정규(Normal)** | 평균 주변 대칭 (예: 측정 오차) | 평균, 표준편차 |
| **삼각(Triangular)** | 최소·최빈·최대를 알 때 (예: 전문가 추정) | min, mode, max |
| **균등(Uniform)** | 범위만 아는 무지 상태 | min, max |
| **로그정규(Lognormal)** | 양수, 우측 꼬리 (예: 매출, 사고 비용) | 평균, 표준편차 (로그 스케일) |

### 시뮬레이션 매뉴얼 (6단계)

1. 결과 변수를 정한다 (예: 순이익).
2. 입력 변수를 모두 적는다 (예: 매출량, 단가, 변동비, 고정비).
3. 각 입력에 분포와 파라미터를 정한다 (전문가 의견 또는 데이터).
4. N=10,000번 시뮬레이션한다.
5. 결과 분포를 본다: 평균, 5%·95% 분위수, 손실 확률 P(이익 < 0).
6. **토네이도 차트**로 어떤 입력이 결과에 가장 영향이 큰지 본다.

### 손실 확률 해석

```
P(이익 < 0)  =  결과 분포에서 이익이 0보다 작은 비율
```

- 5% 미만: 안전한 계획
- 5~20%: 주의가 필요
- 20% 초과: 재설계 필요

### 토네이도 차트

각 입력 변수를 ±1 표준편차씩 흔들었을 때 결과가 얼마나 변하는지 막대로 그린다. 막대가 길수록 그 변수가 **핵심 리스크**다.

### 시나리오 × 시뮬레이션 통합

10주차에서 만든 4개 시나리오마다 입력값 분포를 다르게 잡고 시뮬레이션한다. 시나리오별 결과 분포를 비교하면 전략의 강건성을 정량적으로 본다.

## 바이브 코딩 방식

학생은 코드를 외우지 않는다.

1. 결과 변수와 입력 변수를 자연어로 적는다.
2. AI에게 분포와 파라미터를 추천받는다.
3. AI에게 시뮬레이션 코드를 요청한다.
4. 결과를 분포·구간·손실 확률로 정리한다.

예시 프롬프트:

```text
나는 [정책/사업명]의 5년 누적 순이익을 시뮬레이션하려고 해.
입력 변수와 분포는 아래와 같아:
- 연 매출: 정규(평균 10억, 표준편차 2억)
- 변동비율: 삼각(min 0.4, mode 0.5, max 0.7)
- 연 고정비: 정규(평균 3억, 표준편차 0.5억)
N=10,000번 시뮬레이션하고 평균·5%·95% 분위수·손실 확률을 알려줘.
히스토그램도 흑백으로 그려줘.
```
