# 14주차: 기말 프로젝트 발표·피어 리뷰

## 수업 목표

이번 주차는 새 도구를 배우지 않는다. 한 학기 동안 쌓은 결과물을 **5분 발표**로 압축하고, 동료의 피어 리뷰를 받아 통합 기획서를 마지막으로 다듬는다.

학생은 수업 후 다음을 할 수 있어야 한다.

1. 통합 기획서를 5분 분량으로 압축해 발표할 수 있다.
2. 동료 발표를 정해진 5점 rubric으로 평가할 수 있다.
3. 받은 피드백 중 무엇을 수용·거절할지 판단하고 그 이유를 기획서에 명시할 수 있다.
4. 최종 통합 기획서를 제출할 수 있다.

## 진행 방식

| 시간 | 활동 |
|------|------|
| 0:00-0:15 | 발표 가이드 + 피어 리뷰 rubric 설명 |
| 0:15-2:30 | 학생 발표 (1인당 5분 발표 + 2분 Q&A) |
| 2:30-2:50 | 피어 리뷰 작성 (1인당 3건) |
| 2:50-3:00 | 받은 피드백 통합 워크시트 작성 |

## 피어 리뷰 5점 척도

| 항목 | 평가 |
|------|------|
| 1. 문제 정의 명확성 | 옵션 비교 기준 |
| 2. 도구 적용 정확성 | 5개 도구가 각자 다른 통찰 |
| 3. 결과 통합력 | 충돌 지점 + 통합 논리 |
| 4. 가정·한계 인지 | 핵심 가정 + 깨졌을 때 대응 |
| 5. 발표·시각화 | 슬라이드 제목 = 메시지 |

총 25점, 동료 3명 평균.

## AI 활용 투명성

기말 프로젝트는 AI 도움을 적극 활용하지만, **AI 사용을 숨기지 않는다**. 부록으로 다음을 첨부한다.

- AI에게 보낸 핵심 프롬프트 1~2개
- AI 출력 그대로 사용한 부분
- 사람이 수정·반박한 점
- AI 환각 점검 결과 (가공 인물·미검증 통계 표시)

## 최종 제출물

10~15쪽 통합 기획서 (PDF 또는 노트북).

| 가정·한계 인지 | 4점 |
